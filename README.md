# TravelMateX

TravelMateX is a comprehensive hotel booking web application designed to provide users with a seamless and premium experience for discovering, booking, and managing hotel stays.

## Features

- **Home & Discovery**: Browse available hotels with an intuitive interface.
- **Search Functionality**: Search for hotels by location and dates.
- **Hotel Details**: View comprehensive details about a single hotel, including images, amenities, and pricing.
- **Wishlist**: Save favorite hotels to a personal wishlist for later viewing.
- **Booking & Payment**: Seamless booking flow including payment processing and order summary.
- **User Authentication**: Secure login and registration system.

## Tech Stack

### Frontend
- **React (Vite)**: Fast and modern frontend framework.
- **React Router**: For seamless single-page application navigation.
- **Material UI & Styled Components**: For a responsive, beautiful, and premium user interface.
- **Axios**: For making API requests to the backend.

### Backend
- **Node.js & Express**: Fast and scalable server-side framework.
- **MongoDB & Mongoose**: NoSQL database for flexible data modeling and storage.
- **JWT (JSON Web Tokens)**: Secure user authentication and session management.
- **Crypto-JS**: For secure data encryption and password hashing.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- MongoDB (Local or Atlas instance)

### Installation

1. **Clone the repository** (if applicable):
   ```bash
   git clone <repository-url>
   cd TravelMateX
   ```

2. **Backend Setup**:
   - Navigate to the backend directory:
     ```bash
     cd backend
     ```
   - Install dependencies:
     ```bash
     npm install
     ```
   - Create a `.env` file in the `backend` directory and configure the necessary environment variables (e.g., `PORT`, `MONGO_URI`, `JWT_SECRET`).
   - Start the backend server:
     ```bash
     npm run dev
     ```

3. **Frontend Setup**:
   - Open a new terminal and navigate to the frontend directory:
     ```bash
     cd frontend
     ```
   - Install dependencies:
     ```bash
     npm install
     ```
   - Start the frontend development server:
     ```bash
     npm run dev
     ```

4. **Access the Application**:
   Open your browser and navigate to the URL provided by Vite (usually `http://localhost:5173`).
